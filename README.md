# 🪖 군적금 똑똑이 (Military Savings Calculator)

장병내일준비적금의 복잡한 금리 체계와 정부 매칭지원금을 한눈에 계산해주는 스마트 계산기 및 관리 시스템입니다.

## ✨ 주요 기능

### 1. 스마트 계산기
- **입대일 기반 자동 금리 선택:** 사용자의 입대일(또는 가입일)을 기준으로 해당 시점에 시행 중인 은행별 금리 버전을 자동으로 찾아 계산합니다.
- **실시간 결과 업데이트:** 납입 개월 수와 은행 선택에 따라 원금, 이자, 정부 매칭지원금을 즉시 합산하여 보여줍니다.
- **상세 분석 결과:** 비과세 혜택과 정부 지원금이 포함된 최종 수령액을 카드 형태로 제공합니다.

### 2. AI 기반 은행 추천
- **맞춤형 조합 찾기:** 주택청약 보유 여부, 급여 이체 계획, 사회적 배려 대상자 여부 등을 분석하여 이자 수익이 가장 높은 최적의 은행 조합을 추천합니다.


### 4. 결과 공유 및 저장
- **환경별 최적화:** 
  - **iOS:** 시스템 공유하기를 통해 카카오톡 등으로 즉시 전송 가능.
  - **Android & Windows:** 이미지 파일(.png)로 즉시 다운로드 지원.
- **개인화:** 사용자 이름을 입력하여 본인만의 결과지를 생성할 수 있습니다.

## 🛠 기술 스택

- **Frontend:** React (TypeScript), Vite
- **Styling:** Tailwind CSS, Framer Motion (애니메이션)
- **Icons:** Lucide React
- **Backend/DB:** Supabase (Database, Auth)
- **Deployment:** Vercel (추천)
- **Analytics:** Google Analytics 4 (GA4)
