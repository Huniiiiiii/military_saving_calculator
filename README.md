# 🪖 군적금 똑똑이 (Military Savings Calculator)

장병내일준비적금의 복잡한 금리 체계와 정부 매칭지원금을 한눈에 계산해주는 스마트 계산기 및 관리 시스템입니다.

## ✨ 주요 기능

### 1. 스마트 계산기
- **입대일 기반 자동 금리 선택:** 사용자의 입대일(또는 가입일)을 기준으로 해당 시점에 시행 중인 은행별 금리 버전을 자동으로 찾아 계산합니다.
- **실시간 결과 업데이트:** 납입 개월 수와 은행 선택에 따라 원금, 이자, 정부 매칭지원금을 즉시 합산하여 보여줍니다.
- **상세 분석 결과:** 비과세 혜택과 정부 지원금이 포함된 최종 수령액을 카드 형태로 제공합니다.

### 2. AI 기반 은행 추천
- **맞춤형 조합 찾기:** 주택청약 보유 여부, 급여 이체 계획, 사회적 배려 대상자 여부 등을 분석하여 이자 수익이 가장 높은 최적의 은행 조합을 추천합니다.

### 3. 클라우드 데이터 관리 (Admin)
- **Supabase 연동:** 로컬 파일 방식에서 탈피하여 실시간 데이터베이스(Supabase)를 통해 금리 정보를 관리합니다.
- **시계열 금리 관리:** 은행별로 시행일(`effectiveDate`)에 따른 여러 금리 버전을 유지할 수 있어 과거 가입자와 신규 가입자 모두에게 정확한 정보를 제공합니다.
- **모바일 최적화 어드민:** 휴대폰에서도 간편하게 금리를 수정, 추가, 삭제할 수 있도록 단계별 내비게이션(Drill-down) UI를 지원합니다.
- **안전한 보안 인증:** Supabase Auth를 통한 이메일/비밀번호 로그인을 적용하여 관리자만 안전하게 접근할 수 있습니다.

### 4. 결과 공유 및 저장
- **환경별 최적화:** 
  - **iOS:** 시스템 공유하기를 통해 카카오톡 등으로 즉시 전송 가능.
  - **Android & Windows:** 이미지 파일(.png)로 즉시 다운로드 지원.
- **개인화:** 사용자 이름을 입력하여 본인만의 결과지를 생성할 수 있습니다.

## 🛠 기술 스택

- **Frontend:** React (TypeScript), Vite
- **Styling:** Tailwind CSS, Framer Motion (애니메이션)
- **Icons:** Lucide React
- **Backend/DB:** Supabase (Database, Auth)
- **Deployment:** Vercel (추천)
- **Analytics:** Google Analytics 4 (GA4)

## 🔑 어드민 진입 방법

1. 앱 첫 화면(온보딩)의 가장 하단 **"© 2026 군적금 똑똑이"** 문구를 **연속 5번** 탭합니다.
2. Supabase에서 생성한 관리자 계정으로 로그인합니다.

## ⚙️ 설정 방법 (.env)

프로젝트 루트에 `.env` 파일을 생성하고 아래 정보를 입력해야 합니다.

```env
VITE_SUPABASE_URL=your_supabase_project_url
VITE_SUPABASE_ANON_KEY=your_supabase_anon_key
```
